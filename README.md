# Web服务脚手架部署运行
使用Docker一键部署

## 安装Docker
- https://docs.docker.com/install/linux/docker-ce/ubuntu/
- https://docs.docker.com/docker-for-windows/install/

## 拉取镜像
./pull.sh

## 启动服务
./up.sh

## 查看日志
./logs.sh

## 停止服务
./down.sh
